# Test email corpuses

This directory contains a various email collections that we use for testing. See
individual subdirectories for more information on using each corpus.

List and short description:
- `aero100`: a small, hand-crafted corpus of emails generated from various real-world modern email clients
- `caml-list`: archive of the "caml-list" mailing list, for 30 years between 1992 and 2022
- `dxflrs`: TODO
- `enron`: the [enron email dataset](https://www.cs.cmu.edu/~enron/)
- `epstein`: a subset of the emails of J. Epstein that have been publicly released
- `hamlet`: a synthetic corpus of emails generated by email tools from the Mirage project
- `rfc`, `legacy`, `malformed`, `thirdparty`: emails from https://github.com/stalwartlabs/mail-parser
